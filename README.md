# Laravel CRM System

## Overview
A Customer Relationship Management (CRM) system developed using Laravel and MySQL.

## Features

- User Authentication
- Customer Management
- CRUD Operations
- Dashboard
- Database Integration

## Technologies

- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/Sulaiman600/crm-project.git

cd crm-project

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
