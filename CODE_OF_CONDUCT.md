# Contributor Code of Conduct

## Our Pledge

We, as contributors and maintainers of the **CRM Project**, are committed to creating a welcoming, respectful, and collaborative environment for everyone. We encourage participation without discrimination based on age, gender, ethnicity, nationality, religion, disability, educational background, or level of experience.

## Our Standards

Examples of positive behavior include:

* Using respectful and inclusive language.
* Being courteous and professional.
* Accepting constructive feedback.
* Working collaboratively with team members.
* Supporting and helping other contributors.

Examples of unacceptable behavior include:

* Harassment, discrimination, or bullying.
* Offensive or abusive language.
* Personal attacks or threats.
* Sharing another person's private information without permission.
* Any behavior that creates an unsafe or hostile environment.

## Our Responsibilities

Project maintainers are responsible for ensuring that this Code of Conduct is followed. They may review, edit, or remove contributions that violate these guidelines and take appropriate action when necessary.

## Scope

This Code of Conduct applies to all project spaces, including the GitHub repository, issues, pull requests, discussions, documentation, and any communication related to the project.

## Enforcement

If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers through the GitHub repository or the official communication channel. All reports will be handled respectfully and confidentially whenever possible.
